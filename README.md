# Anuditya Gautam · Portfolio

A modern, responsive portfolio website for a 2nd-year Computer Science Engineering student. Built with HTML, CSS, and JavaScript – no frameworks, no backend.

## 🔗 Live Demo
[Deploy on Vercel](https://your-vercel-url.vercel.app)

## ✨ Features
- Dark theme with glassmorphism cards
- Smooth scrolling and scroll progress bar
- Typing animation for hero name
- Floating background blobs
- Project cards with live demo and GitHub links
- Animated skill cards
- Education timeline
- Currently learning section with flip animations
- Achievements with icons
- Contact cards with email, GitHub, LinkedIn, and resume download
- Back to top button
- Fully responsive (mobile, tablet, desktop)
- Loading screen
- Button ripple effect

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, animations, glassmorphism)
- JavaScript (vanilla)
- Font Awesome 6
- Google Fonts (Poppins)
- AOS (Animate On Scroll)
